Sistema de Design de Jogos Educativos

Uma abordagem com multiagentes baseados em LLMs para o design de jogos educativos

Inteligência Artificial

Sistema integrado com LLMs (Large Language Models) que atuam como agentes especializados para auxiliar no processo de design de jogos educativos.

Multiagentes

Diferentes agentes especializados trabalham em conjunto: agente de brainstorming, agente socrático, especialista em Bloom e agentes de design.

Processo de Design
1. Brainstorming
Geração de ideias com auxílio de IA
2. Reflexão Socrática
Análise crítica das ideias
3. Taxonomia de Bloom
Definição de objetivos educacionais
4. Game Design Canvas
Estruturação do projeto
Método Socrático

Aplicação de questionamentos estruturados para reflexão crítica sobre as ideias geradas.

Taxonomia de Bloom

Classificação automática de objetivos educacionais nos 6 níveis cognitivos.

Endo-GDC

Game Design Canvas especializado para jogos educativos endógenos.
